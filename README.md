# Conduit API Testing (Postman)

This repository contains my API testing project for the Conduit application using Postman.

---

## 📌 Project Overview

The goal of this project was to test REST API endpoints of the Conduit application and validate core functionalities such as authentication, user management, and article operations.

---

## 🧪 Scope of Testing

### User
- registration
- login
- get current user
- update user data

### Articles
- create article
- update article
- delete article
- get articles
- feed & filtering

### Profile
- get profile
- follow / unfollow user

### Comments
- add comment
- delete comment
- validation scenarios

### Tags
- retrieve tags list

---

## 🔍 What Was Tested

- status codes (200, 401, 403, 404, 422)
- response structure
- authentication & token handling
- positive scenarios
- negative scenarios
- error handling

---

## 🛠 Tools & Technologies

- Postman
- REST API
- JSON
- GitHub

---

## 📂 Repository Structure

- `/collections` → Postman collection  
- `/docs` → API documentation and test scenarios  
- `/tests` → sample Postman test scripts  
- `/reports` → test summary  
- `/screenshots` → request/response examples
